# NextDesk
A modern landing page for NextDesk, a coworking space with a built-in chatbot for instant support.

# Technologies
 ReactJS, Typescript, TailwindCSS, Shadcn, Tidio, Motion,  hosted and optimized via Cloudflare CDN.
